## 5.1 BASIC IDENTITIES 基本恒等式
符号 $\dbinom{n}{k}$ 是一个二项式系数，之所以这样称呼，是因为本节后面将会看到的一个重要性质——二项式定理。不过我们把这个符号读作“$n$ 选 $k$”。这个叫法源自它的组合意义：它表示从一个含 $n$ 个元素的集合中选出 $k$ 个元素的子集的方法数。例如，从集合 $\{1,2,3,4\}$ 中选出两个元素，共有六种方法。
```math
\{1,2\},\ \{1,3\},\ \{1,4\},\ \{2,3\},\ \{2,4\},\ \{3,4\};
```
所以 $\dbinom{4}{2} = 6$。
